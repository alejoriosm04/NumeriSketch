Ejemplos permitidos

Tenga en cuenta que para operar una constante por una variable debe usar el operador "*" Ejemplo: 3*x

A continuación, se muestran algunos ejemplos de funciones válidas que puedes usar:

Ejemplos para f(x):

  • sin(x) + x^2 - 3*x
  • cos(x) - x^2 + 2*x
  • log(x + 1) + sqrt(x)
  • exp(x) - pi*x

Ejemplos para g(x):

  • (x^2 + 3)/(2*x)
  • sqrt(x + 2)
  • (sin(x) + 1)/x
  • (exp(x) - 1)/(2 + x)
